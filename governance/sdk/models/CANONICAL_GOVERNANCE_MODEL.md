# Canonical Governance Model

Status: ACTIVE / AUTHORIZED FOR IMPLEMENTATION

## ValidationContext

Carries baseline, repository, execution, and compatibility metadata.

## ValidationResult

Normalized validator result containing validator, status, severity, finding, recommendation, evidence, and timestamp.

## ValidationReport

Aggregated compliance report generated from validation results.

## GovernanceEvidence

Structured evidence used for audit and certification.

## ComplianceStatus

Allowed values: PASS, WARNING, FAIL, NOT_APPLICABLE.
