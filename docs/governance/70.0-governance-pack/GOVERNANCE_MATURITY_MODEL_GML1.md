# Governance Maturity Model — GML-1

## Status
APPROVED

## Level
GML-1 — Foundation Governance

## Objective
GML-1 certifies that DentalOperix has consolidated documentary governance, a single active technical baseline, master registers, traceability, protected component control, and a mandatory architecture/governance review process before implementation.

## Certification Criteria

1. Single active technical baseline exists.
2. Historical baselines are preserved as audit evidence.
3. Governance Master Index is approved.
4. Document hierarchy is defined.
5. Document Registry exists.
6. Certification Register exists.
7. Baseline Transition Register exists.
8. Governance Change Log exists.
9. Certified architectures and Sources of Truth are documented.
10. Permanent restrictions and protected components are documented.
11. Every initiative requires architectural analysis, dependency review, risk review, technical impact, baseline compatibility review, governance determination, and approval before implementation.
