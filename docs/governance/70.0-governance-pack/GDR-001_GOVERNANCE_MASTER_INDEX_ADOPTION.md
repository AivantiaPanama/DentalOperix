# GDR-001 — Governance Master Index Adoption

## Status

APPROVED

## Date

2026-06-24

## Decision

DentalOperix adopts the Governance Master Index (GMI) as the highest-level governance control document for program documentation.

The GMI does not replace the active technical baseline. It governs, references, and consolidates the official program state.

## Active Technical Baseline

DENTALOPERIX_BASELINE_69_2

## Historical Baseline Treatment

Baseline 57.9 remains certified historical evidence and is superseded as the active documentary reference by Baseline 69.2.

## Governance Impact

- No source code impact.
- No architecture impact.
- No Source of Truth change.
- No protected component modification.
- No persistence re-architecture.

## Required Maintenance Rule

Every major phase closure must update the GMI and applicable governance registers before being considered formally closed.
