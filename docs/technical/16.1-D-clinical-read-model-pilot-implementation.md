# 16.1-D — Clinical Read Model Pilot Implementation

## Resultado

Se implementa el piloto inicial del dominio Clinical como consumidor read-only de Read Models.

## Alcance incluido

- TreatmentPlans
- TreatmentStages
- ClinicalOutcomes

## Alcance excluido

- ClinicalNotes
- Escritura clínica
- Migración productiva
- Dual write

## Arquitectura

```text
ReadModelSourceProvider
        ↓
ClinicalAggregateReadService
        ↓
TreatmentPlanReadAdapter
TreatmentStageReadAdapter
ClinicalOutcomeReadAdapter
        ↓
Clinical Read Models
        ↓
Fallback Lead Projection
```
