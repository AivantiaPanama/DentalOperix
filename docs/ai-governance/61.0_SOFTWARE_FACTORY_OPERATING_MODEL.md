\# 61.0 Software Factory Operating Model



Status: ACTIVE



Owner: Product Governance



\## Purpose



Define the official operating model used by DentalOperix for AI-assisted software delivery.



This document is authoritative for all contributors, AI systems, reviews, and implementation activities.



\---



\# Development Lifecycle



Idea

→ GitHub Issue

→ Iteration Definition

→ AI Task Assignment

→ Branch Creation

→ Implementation

→ Review

→ Pull Request

→ Merge

→ Documentation Update



\---



\# Official Workflow



\## Step 1



Create GitHub Issue.



Must include:



\* Objective

\* Business value

\* Scope

\* Priority



\---



\## Step 2



Create Iteration Document.



Location:



docs/iterations/



Example:



ITERATION\_61.1\_USERS\_RBAC.md



\---



\## Step 3



Create AI Task Package.



Location:



docs/ai-tasks/



Examples:



\* CHATGPT

\* CLAUDE

\* CURSOR

\* V0



\---



\## Step 4



Assign Work



ChatGPT:



\* Architecture

\* Governance

\* Technical Review



Claude:



\* Documentation

\* Requirements

\* User Stories



Cursor:



\* Implementation

\* Refactoring

\* Tests



v0 / Figma:



\* UX

\* Dashboards

\* UI Components



\---



\## Step 5



Create Branch



Pattern:



ai/<platform>-<initiative>



Example:



ai/cursor-61.1-users-rbac



\---



\## Step 6



Execute Work



Output must be stored in:



docs/ai-outputs/



\---



\## Step 7



Review



Required:



\* Architecture Review

\* Governance Review

\* Technical Review



\---



\## Step 8



Create Pull Request



Must use:



.github/PULL\_REQUEST\_TEMPLATE.md



\---



\## Step 9



Merge



Only after approval.



\---



\## Step 10



Update Product Memory



Required updates:



\* CURRENT\_PROJECT\_STATUS.md

\* PRODUCT\_MEMORY.md

\* PRODUCT\_DECISION\_LOG.md (if applicable)



\---



\# Mandatory Context



All AI systems must receive:



1\. DENTALOPERIX\_MASTER\_CONTINUATION\_PROMPT.md

2\. DENTALOPERIX\_QUICK\_START.md

3\. DENTALOPERIX\_AI\_CONTEXT.md

4\. Current Iteration Document



before executing work.



\---



\# Governance Rules



Must preserve:



\* Leads = Source of Truth

\* Certified Architecture

\* Product Governance

\* AI Governance



No implementation may bypass these rules.
