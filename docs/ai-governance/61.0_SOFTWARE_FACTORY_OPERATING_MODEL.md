\# Multi-AI Operating Model Reference

Authoritative companion document:

```text
docs/product-governance/61.0_MULTI_AI_OPERATING_MODEL.md
```

The Software Factory is operated through a coordinated multi-AI model:

\* ChatGPT: Architecture, governance, review, certification and AI delivery coordination. \* Claude: Functional documentation, user stories, acceptance criteria and business rules. \* Cursor: Controlled implementation, refactoring, tests and pull request preparation. \* v0 / Figma: UX, wireframes, dashboard layouts and visual component proposals.

No AI system may skip architecture approval, implementation approval, pull request review, documentation update, or release readiness review.

The multi-AI model does not modify certified architecture and must preserve Leads = Source of Truth.
