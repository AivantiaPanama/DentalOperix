# v0 Dashboard Prompt

Use this prompt for v0 dashboard and UI mockups.

```text
Design a DentalOperix dashboard for the assigned role.

Use the provided dashboard spec and product context.

Return React UI mockup only.
Do not implement persistence.
Do not include business logic.
Do not call APIs.
Focus on clarity, actionability, responsive layout, and dental clinic operations.

Dashboard should answer:
- What is happening?
- What requires attention?
- What action should the user take next?
```
