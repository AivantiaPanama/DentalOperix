# Domain Analysis Framework

**Generated:** 2026-06-25  
**Framework:** DentalOperix Engineering & Governance Framework  
**Status:** Reusable framework component.

## Purpose

Reusable method for analyzing a domain before implementation.

## Required Outputs

- Domain purpose and ownership.
- Bounded-context classification.
- Ubiquitous language.
- External dependencies.
- Source-of-truth interaction model.
- Out-of-scope exclusions.
- Governance risks.
- Certification recommendation.

## Reuse Rule

A new domain fills in domain-specific facts. The analysis structure is not recreated.
