## Purpose

Reusable method for validating conformance to the certified architecture.

## Required Checks

- Baseline compatibility.
- Source-of-truth preservation.
- Bounded-context boundary preservation.
- Protected component impact.
- Persistence pattern compatibility.
- No Dual Write.
- No Persistence Re-Architecture.
- Incremental and traceable change scope.

## Program 75.x Interpretation

Work package reviews are conformance reviews. They do not redesign architecture unless formal evidence demands it.
