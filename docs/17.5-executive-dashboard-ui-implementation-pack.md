## Architectural decision

17.5 introduces presentational UI components only. It does not bind routes, does not implement fetch, does not modify login and does not create or expose new APIs.

Allowed flow:

Dashboard UI → 17.4 UI Foundation Pack → 17.3 Contracts → ExecutiveDashboardApiContracts

Forbidden flows remain blocked:

- UI → Aggregates
- UI → Adapters
- UI → Raw Telemetry
- UI → Read Sources
- UI → Direct Read Models
