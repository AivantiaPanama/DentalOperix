# DentalOperix Product Context

Status: ACTIVE
Owner: Product Governance
Last updated: 2026-06-20

## Product Definition

DentalOperix is a clinical-commercial operating system for dental clinics. It should help clinics capture patients, schedule appointments, operate the daily agenda, follow up, analyze results, and eventually deliver clinical intelligence.

## User Profiles

### Patient

Needs:

- View upcoming appointment.
- Receive confirmation and calendar invite.
- Manage contact and billing data in future.
- View history in future portal.

### Assistant

Needs:

- Operate daily agenda.
- Confirm appointments.
- Contact pending patients.
- Track no-shows and reschedules.
- View new leads.

### Doctor/Dentist

Needs:

- View agenda.
- View assigned patients.
- See relevant patient history.
- Add clinical notes in future scope.

### Administrator

Needs:

- View executive KPIs.
- Manage users and roles.
- Review conversion and no-shows.
- Monitor clinic performance.

## Sellable Starter Focus

The earliest sellable version should prioritize:

- Lead capture.
- Appointment scheduling.
- Patient notifications.
- Clinic calendar.
- Assistant Dashboard.
- Admin Starter Dashboard.
- Users and RBAC.
