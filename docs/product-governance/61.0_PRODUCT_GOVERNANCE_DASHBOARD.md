# 61.0 Product Governance Dashboard

Status: ACTIVE
Owner: Product Governance
Last updated: 2026-06-20

## Purpose

This dashboard tracks product readiness, business value, priority, dependencies, and estimated progress. It is not a runtime dashboard; it is a governance artifact for deciding where to invest next.

## Status Vocabulary

| Status | Meaning |
|---|---|
| NOT_STARTED | No active design or implementation |
| PLANNED | Approved for future design |
| IN_DESIGN | Requirements or architecture are being defined |
| READY_FOR_AI | Task package is ready for external AI execution |
| IN_PROGRESS | Implementation or design is active |
| VALIDATION | Tests/manual validation in progress |
| CERTIFIED | Accepted under DentalOperix governance |
| PRODUCTION | Operationally validated in running environment |

## Priority Vocabulary

| Priority | Meaning |
|---|---|
| P0 | Required to sell or safely operate Starter |
| P1 | Required for professional clinic operation |
| P2 | Optimization, analytics, or automation |
| P3 | Future/enterprise expansion |
