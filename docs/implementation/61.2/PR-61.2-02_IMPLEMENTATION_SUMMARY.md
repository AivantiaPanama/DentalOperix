## Implementation notes

- Added a read-only Today's Schedule widget.
- The widget lists today's non-cancelled appointments ordered by start time.
- The widget displays patient name, appointment time, service, and provider display label.
- The widget shows an explicit empty state when no appointments exist for today.
- No inline cancel/delete/edit actions were introduced.
- No Appointment Detail, create, reschedule, or cancel workflow was introduced in this PR.
- No protected components were modified.
