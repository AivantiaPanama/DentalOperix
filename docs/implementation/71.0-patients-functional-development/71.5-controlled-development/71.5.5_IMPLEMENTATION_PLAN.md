## 4. Risk Controls

| Risk                            | Control                                       |
| ------------------------------- | --------------------------------------------- |
| Query duplication               | Centralize query logic in PatientReadService. |
| API-to-persistence bypass       | Architecture guards and review.               |
| Write side effects during reads | Explicit read-only service contract.          |
| DTO inconsistency               | Single DTO contract source.                   |
| Regression in Patient API       | Endpoint tests plus full regression suite.    |
