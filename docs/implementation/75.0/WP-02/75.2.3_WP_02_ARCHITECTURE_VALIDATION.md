# 75.2.3 — WP-02 Architecture Validation

## Determination

WP-02 is compatible with the certified Clinical Records architecture from WP-01.

## Aggregate root

- ClinicalRecord.

## First entity

- ClinicalNote.

## Value objects

- ClinicalNarrative.
- HealthcareProfessionalId.

## Domain boundaries

Clinical Records owns clinical information only.

It may reference Patients and Appointments, but it must not mutate them.
