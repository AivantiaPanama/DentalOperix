---

## 1. Objective

Implement the Patient Application Services layer on top of the certified Patient Foundation.

The implementation must:

```text
Preserve Domain Authority
Preserve Repository Authority
Preserve Lead Authority
Remain UI Independent
```

The implementation must not:

```text
Introduce UI
Introduce Dual Write
Modify Lead Persistence
Bypass Repositories
```

---

## 2. Scope

In scope:

```text
PatientCreationService

PatientMergePreparationService
PatientAuditService
```

Out of scope:

```text
Patient UI

Patient Merge UI
```
