## Implemented Changes

### Security containment

- Removed real `.env.relational.prod` from the working tree.
- Kept only example/template environment files.
- Recommendation remains: rotate any secrets if the original package was shared externally or committed.

### Runtime persistence alignment

- `LeadPersistenceProvider` now defaults to `relational-db`.
- Google Sheets adapter is restricted to explicit rollback mode only:

```text
LEADS_PERSISTENCE_MODE=google-sheet
GOOGLE_SHEETS_ROLLBACK_APPROVED=true
```

- Lead creation and post-creation status updates now use `LeadPersistenceProvider`.
- `/api/leads/list` now reads through `LeadPersistenceProvider`.
- Derived analytics and operational read paths now read through `LeadPersistenceProvider`.
- No dual write was introduced.
- No new source of truth was introduced.
