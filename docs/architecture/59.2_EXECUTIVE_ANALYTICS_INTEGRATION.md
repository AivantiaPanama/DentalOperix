- `src/routes/admin/dashboard.tsx`

## Runtime Flow

```text
Admin Dashboard
  -> fetchRevenueDashboardMetrics(period)
  -> fetchExecutiveAnalytics(period)
  -> ExecutiveAnalyticsPanel
```

Executive Analytics remains an additional read-only panel. It does not replace the 58.x Revenue Intelligence dashboard metrics.

## Dashboard Content

The panel displays:

- Revenue Score
- Growth Score
- Opportunity Index
- Source Ranking
- Service Ranking
- Executive Alerts
- Executive Opportunities
- Governance marker: Source of Truth = Leads

## Failure Behavior

Executive Analytics failures are isolated from the main dashboard.

If `/api/analytics/executive` fails, the dashboard still renders Revenue Intelligence metrics and the Executive Analytics panel shows a scoped warning.

## Governance Notes

The historical `src/components/admin/executive-dashboard/*` package remains untouched because it belongs to a prior controlled package with explicit constraints.

59.2 uses a new, isolated component instead.
