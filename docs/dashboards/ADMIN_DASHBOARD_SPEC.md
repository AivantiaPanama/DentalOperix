# Admin Dashboard Specification

Status: DRAFT
Owner: Product / UX Governance
Last updated: 2026-06-20

## Goal

Help clinic leadership understand performance and decide where to act.

## Primary Questions

- How many leads are arriving?
- How many appointments are scheduled?
- How many patients attend?
- What is the no-show rate?
- Which channels and services convert best?
- Is the clinic operating at capacity?

## Core Components

1. Executive KPI Strip
2. Conversion Funnel
3. Appointment Attendance Summary
4. Lead Source Performance
5. Service Demand View
6. Operational Alerts

## Data Rules

- Analytics is read-only.
- No dashboard write-back.
- Data quality flags must be visible when relevant.
